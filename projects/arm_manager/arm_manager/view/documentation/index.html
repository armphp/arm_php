Acura is a modern and elegant Real Admin Template! It's very clean and easy to customize. It come with many plugins and useful widgets. It's well organized and it contain many basics options, where you can make your modification, or add your taste!

File Structure

The Structure of Acura template is very clean:

  • CSS: The LessCSS file and the compiled and minified CSS files.
  • JS: It contain the plugins and the core acura.js file.
  • Media: It contain the images, like the logo image.
  • HTML: It come with the 15 HTML pages.
  • Documentation: What you read now.
  • Design: The basic PSD layout.

HTML Hierarchy

Sidebar Type

You can chose one from the 2 type of the sidebars come with Acura. If you want the large sidebar it's already the default. and if you want the reduced one, just add the sidebar-reduce class to the aside.sidebar element, as below:

<div class="sidebar sidebar-reduce">
  <!-- Sidebar Data-->
</div>

Grids

Acura used the grid of 12 columns, you have only to add the class grid-X, where X is a number between 1 and 12. For ex:

<!-- Grid 7 Columns -->
<div class="grid-7"></div>
<!-- Grid 11 Columns -->
<div class="grid-12"></div>

Buttons

The basic structure of a button is:

<!-- Div element button -->
<div class="btn">TEXT</div>
<!-- Link element button -->
<a href="#" class="btn">TEXT</a>

Below some useful class you can add them to the btn class:

  • btn-badge: A button not clickable. You can use it like a badge for example.
  • btn-border: Button with border.
  • btn-transparent: Button with a transparent background.
  • btn-o-icon: In case you want to use a button icon only.
  • btn-block: Block button (Full width).
  • btn-disabled: Buttons disabled.
  • btn-flat-3d: button flat with a 3D style.
  • btn-3d: A 3D button style.
  • btn-bottom-bar: A button with bottom border hover style.
  • btn-top-bar: A button with top border hover style.
  • btn-left-bar: A button with left border hover style.
  • btn-right-bar: A button with right border hover style.
  • btn-square: A squared button style.
  • btn-error: An error button style.
  • btn-warning: A warning button style.
  • btn-info: An info button style.
  • btn-submit: A submit button style.
  • btn-success: A success button style.
  • btn-extra-large: An extra large button.
  • btn-large: A large button.
  • btn-small: A small button.
  • btn-tiny: A very small (tiny) button.

Of course, you can make a combination with the class above. Some combination maybe not go together.

If you want to make a grouped buttons, below is the structure: 

<div class="btn-group">
  <!-- Buttons 1 -->
  <!-- Buttons 2 -->
  <!-- Buttons 3 -->
</div>

And if you want a justified grouped buttons, just add the group-justified class to the div.btn-group element.

Icons

To use some icons in your text just add <i class="i">UNICODE</i>.

For putting icon on the left of a text add the class i-left for making some space. If the icon are on the right side add the class i-right.

If you want a small icon add the class i-small, and if you need a big one, add i-big.

If you are looking for the UNICODE of icons, just check the Icons page, or check this link here.

Menu

Below you see the structure of a menu with full options:

<div class="menu">
  <ul>
    <!-- Standard item -->
    <li>
      <a href="#">
        <div class="menu-icon"><!-- Icon Unicode --></div>
        <div class="menu-title">
          Title
          <!-- Count number -->
          <span class="menu-count">10</span>
        </div>
      </a>
    </li>
    <!-- Disabled item -->
    <li>
      <a href="#" class="disabled">
        <div class="menu-icon"><!-- Icon Unicode --></div>
        <div class="menu-title">
          Disabled
        </div>
      </a>
    </li>
    <!-- Separator -->
    <li class="separator"></li>
    <!-- Item Not Link -->
    <li>
      <div class="menu-item">
        <div class="menu-icon"><!-- Icon Unicode --></div>
        <div class="menu-title">
          Not Link
        </div>
      </div>
    </li>
  </ul>
</div>
